How the suggested action is decided
Combines your profit/loss with the advisor signal:
  • SELL → take profit if up, cut if down.
  • TRIM → lock some profit if up ≥20%; review if near stop.
  • BUY/ACCUMULATE → add (esp. near/below cost); let winners run, add on dips.
  • HOLD → hold; consider trimming a gain ≥30%; review if down ≥8% (near stop-loss).
